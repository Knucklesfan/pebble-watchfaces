# pebble-watchfaces
A collection of Pebble Time 2-focused watchfaces.

# Compilation tips
- Make sure you have the pebble sdk installed
- Make sure you're using emery (i havent tested other platforms)
- Make sure to run `bear -- pebble build` to make clangd work
- `pebble build` and `pebble install --emulator emery` are your best friends
- good luck
